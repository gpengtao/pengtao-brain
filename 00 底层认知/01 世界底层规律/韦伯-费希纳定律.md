# 韦伯-费希纳定律

## 1. 本质定义

人类感官对刺激的**感知强度**，与刺激的**物理强度的对数**成正比——大脑天生用对数方式处理外部世界。

```
感知强度 ∝ log(物理刺激强度)
```

## 2. 核心金句

- **大脑不是线性感知机器，是对数感知机器**
- **声音响一倍，感觉不是变了一倍**
- **边际递减不只是心理现象，是神经生理层面的硬编码**
- **为什么有钱了还不快乐？因为大脑对财富的感知本来就是对数的**

## 3. 内在机理

**进化压力下的最优编码方案**
大脑需要用有限的神经资源，同时感知极弱到极强的刺激（从一根蜡烛到烈日）。对数压缩是解决这一矛盾的最优解——压缩动态范围，保留辨别能力。

> 这是几亿年进化筛选出来的感知架构，不是后天习得的心理倾向。

**两人分阶段建立：**
韦伯发现"刚好可觉察的差异"与刺激强度成正比（`ΔS/S = k`），费希纳在此基础上积分得到对数关系。

## 4. 核心特征

- **公式**：`P = k × log(S/S₀)`（P 感知强度，S 物理刺激，S₀ 感知阈值）
- **曲线形态**：对数曲线——低刺激段感知变化显著，高刺激段趋于钝化
- **普适性**：听觉、视觉、触觉、痛觉、财富感知，全部服从同一规律
- **关键含义**：物理刺激翻倍，感知只增加一个固定量（log 的性质）

## 5. 适用场景

| 领域 | 具体表现 |
|------|---------|
| **听觉** | 声音从20%调到40%感觉明显；从80%调到100%几乎感觉不出 |
| **视觉** | 一根蜡烛→两根，感觉很亮；100瓦→200瓦，几乎无感 |
| **财富幸福** | 穷→小康幸福飙升；富→更富幸福几乎不涨 |
| **学习进步** | 入门期进步感强烈；精通期进步感极弱（哪怕绝对提升相同） |
| **产品设计** | 用户对变化的感知是对数的，改版要超过"感知阈值"才有效 |

## 6. 具象案例

- **音量**：耳机从20%调到40%，感觉明显变响；从80%调到100%，几乎无变化
- **涨薪**：月薪5000涨到8000，感觉翻天覆地；月薪5万涨到8万，感觉还好
- **举重**：手里空着加1公斤感觉很重；已举10公斤时加1公斤几乎感觉不到

## 7. 适用边界 & 常见误区

- **误区1：这只是心理作用**——不是，这是可测量的神经电信号规律，有物理基础
- **误区2：是绝对准确的定律**——史蒂文斯幂次定律在某些场景下更精确，韦伯-费希纳是简化近似
- **不适用**：极端刺激强度下（接近痛觉阈值），对数关系会失效

## 8. 个人行动指导

1. **理解幸福感的天花板**：收入到一定程度后感知增量极小，别用钱填不该用钱填的需求
2. **学习曲线利用**：入门阶段反馈最强，用这个"高反馈期"建立学习习惯和信心
3. **情绪管理**：和谁比决定了你对同等收入的感知，主动选择参照系
