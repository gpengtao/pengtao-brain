# 沉没成本谬误

## 1. 本质定义

因为已经投入了时间、金钱或精力，而继续一件本来不值得继续的事——**已经发生、无法收回的成本（沉没成本）不应该影响未来决策**，但大脑天然会把它算进去。

## 2. 核心金句

- **已经花出去的钱，不是继续花的理由**
- **问自己：如果今天是第一天、零成本，我还会做这个选择吗？**
- **放弃不是失败，是把资源投向更有价值的地方**
- **继续的代价是未来，不只是过去的损失**

## 3. 内在机理

**[[损失厌恶]]是核心驱动**
放弃 = 在心理上"确认损失"。损失厌恶让大脑极度抗拒承认损失，于是选择继续（哪怕继续也会亏），至少不用在当下"确认"。

**身份认同防御**
承认沉没成本是错误 = 承认自己判断失误 = 自我形象受损 → [[确认偏误]]介入，主动寻找"继续合理"的理由。

**心理账户（Mental Accounting）**
大脑把钱分进不同"账户"，已花的那笔如果不继续，账户就"亏本关账"了——心理上难以接受。

## 4. 核心特征

- **理性原则**：决策只应基于**未来的成本和收益**，沉没成本与未来无关
- **谬误的表现**：把已花的成本偷偷加进去，扭曲了对"继续还是放弃"的判断
- **机会成本被忽视**：继续一件不值得的事，代价是本可以做更值得事情的时间和精力
- **越投入越难放弃**：已投入越多，损失厌恶 + 确认偏误双重加强，越难做出理性判断

## 5. 适用场景

| 领域 | 沉没成本谬误表现 |
|------|--------------|
| **项目管理** | 项目明显失败，因"已投入6个月"继续推进 |
| **职业选择** | 工作不适合，因"已经做了3年"不愿离职 |
| **投资** | 股票亏损，因"我买的时候100块"死扛不割 |
| **关系** | 明知不健康，因"在一起这么多年"不愿放手 |
| **日常消费** | 餐厅食物难吃，因"点都点了"硬吃完 |

## 6. 具象案例

- **电影**：花200元买票，看了30分钟觉得很烂——理性选择是离开（200元已沉没，继续只是再浪费90分钟）
- **创业**：公司烧了500万，失败信号明显——沉没成本让创始人继续烧，直到耗尽所有资源
- **健身卡**：办了年卡不适合，因为"卡还有半年"不愿换——结果半年没去一次

## 7. 适用边界 & 常见误区

- **误区1：放弃沉没成本就是轻易放弃**——不是；基于未来价值的坚持是合理的，基于过去成本的坚持是谬误
- **误区2：所有"坚持"都是沉没成本谬误**——如果继续有真实的未来价值（积累效应、关系深化），坚持是合理的
- **误区3：只要亏损就该止损**——止损也要看未来：如果反转概率高，坚持可能是对的；关键是**基于未来，不是基于过去**
- **不适用**：声誉、信用类资产——"已经投入的声誉"确实影响未来（这不是谬误，是真实的机会成本）

## 8. 个人行动指导

1. **"今天是第一天"测试**：如果今天才开始、没有任何历史投入，我还会做这个选择吗？
2. **分离"已花的"和"将花的"**：明确列出未来还需投入多少、未来能得到什么，只用这两个数字决策
3. **预设止损线**：提前写下"什么情况下放弃"，执行时不靠当下情绪

## 9. 关联模型

- [[损失厌恶]] — 核心驱动：放弃 = 在心理上确认损失
- [[确认偏误]] — 投入越多，越倾向于为"继续"找合理化理由
- [[锚定效应]] — 历史成本成为"值不值"的判断锚点
- [[边际递减效应]] — 继续投入边际收益可能已极低，但沉没成本让人感知不到
