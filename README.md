# pengtao-brain

个人认知知识库，使用 Obsidian 管理。详见 [index.md](index.md)。

## 新机器初始化

1. clone 本仓库，用 Obsidian 打开为 vault
2. 手动安装以下社区插件（Settings → Community plugins）：
   - **Dataview** — 动态查询笔记
   - **Folder notes** — 文件夹的 index.md 自动关联
   - **Homepage** — 打开 vault 自动跳转 index.md
   - **Tag Wrangler** — 标签批量管理
   - **Tasks** — 任务管理
   - **Mind Map** — 思维导图视图
   - **Terminal** — 内置终端
